# Chaudhary Health Care Center - Koraon, Prayagraj

A web application for the Chaudhary Health Care Center located in Koraon, Prayagraj.

## Features
- Patient Management
- Billing System
- Daily Notes
- Responsive Design

## Project Structure
- `index.html`: Main landing page
- `css/`: Stylesheets
- `js/`: JavaScript logic for billing, patients, and main site functionality
- `hlogo.png`: Hospital logo
